# Emotion-Detection-in-Tweets
Project with multiple objectives around identifying emotions in a dataset of tweets.


## Objectives
1.	Evaluate and anotate a dataset and create an Inter-Annotator Agreement analysis.
2.	Train models (i.e. MLP, Naïve Bayes, SVM) for tweet classification – with pre-processing, text representation (i.e. SentenceTransformer, TF-IDF), and evaluation with metrics (Precision, Recall, F1-Score e Accuracy).
3.	Classifying tweet emotions with LLMs, comparing different models, hyperparameters, and Prompt Engineering techniques.
